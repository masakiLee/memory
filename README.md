# 日日背單字 - 日日是好日 (2022 切版直播班夏季專題)
### 專案緣由
日常生活中無法自由運用是學習外語的一大瓶頸，其中很大的原因來自單字量不足。這款軟體可以因應每位使用者的學習習慣，陪伴日常學習。每日背單字，進步每日見。
```
 - 使用 ADOBE XD 自行設計 Layout
 - 使用 Bootstrap5 & SCSS 進行切版
 - 使用 Gulp 編譯工具
```
